Overview

A compact personal business card website built to practice HTML/CSS layout, visual hierarchy and simple responsive presentation.

Responsive Styling

Practiced spacing, typography, color and sizing so the card works cleanly across browser widths.

Published the static site through Netlify from a GitHub repository to make the project easy to share.
