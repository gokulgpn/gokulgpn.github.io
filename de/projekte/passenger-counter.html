Fahrgastzähler

JavaScriptViteÖPNVNetlify

Überblick

Eine Fahrgastzähler-App, inspiriert von Münchner U-Bahn-Anwendungsfällen, entwickelt zum Üben von Inkrementierung, Speicherung und Zurücksetzen von Werten in JavaScript.

Was ich gebaut habe

Zählerlogik

Aufbau von Inkrementierungs- und Speicher-Aktionen, damit Fahrgastzahlen während wiederholter Zählsitzungen erfasst werden können.

ÖPNV-Kontext

Verwendung des U-Bahn-Szenarios als praktisches Thema, das die JavaScript-Übung mit Interessen aus dem Bereich Verkehr und Mobilität verbindet.

Vite + Netlify

Strukturierung des Projekts mit Vite und Deployment des GitHub-Repos auf einer öffentlichen Netlify-Website.

Tools & Technologien

  • JavaScript für Zählerinteraktionen
  • Vite für die lokale Entwicklungsumgebung
  • HTML und CSS für die App-Oberfläche
  • GitHub und Netlify für die Veröffentlichung
