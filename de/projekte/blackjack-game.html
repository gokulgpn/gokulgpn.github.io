Blackjack-Spiel

JavaScriptSpiellogikHTMLNetlify

Überblick

Ein browserbasiertes Blackjack-Spiel, entwickelt zum Üben von JavaScript-Grundlagen über eine kleine, spielbare Oberfläche.

Was ich gebaut habe

Spielablauf

Implementierung des Spielstart- und Karte-Ziehen-Ablaufs, Verfolgung der Karten, des Punktestands und des Rundensstatus des Spielers.

Punktelogik

JavaScript-Konditionale und Funktionen zur Berechnung der Kartensummen und Anzeige von Gewinn-, Verlust- oder Weiterspielen-Meldungen.

Live-Deployment

Veröffentlichung des Projekts von GitHub auf Netlify, sodass es direkt im Browser geöffnet und gespielt werden kann.

Tools & Technologien

  • JavaScript für Spiellogik und DOM-Updates
  • HTML und CSS für die Spieloberfläche
  • GitHub für Versionskontrolle
  • Netlify für Live-Deployment
