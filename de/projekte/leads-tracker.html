Leads Tracker — Chrome-Erweiterung

JavaScriptChrome-ErweiterungWebentwicklungOpen Data

Überblick

Eine Chrome-Erweiterung zum Speichern und Organisieren nützlicher Datensatz-URLs bei der Recherche europäischer offener Datensätze. Das Projekt entstand im Rahmen des JavaScript-Lernens über Scrimba — Kombination aus Kursinhalten und einem praktischen Anwendungsfall: öffentlich lernen und bauen.

Motivation

Bei der Erkundung europäischer offener Datenquellen für GIS- und Raumanalysearbeit wurde die Verwaltung Dutzender nützlicher Datensatz-URLs über Tabs hinweg mühsam. Der Leads Tracker löst dieses Problem, indem URLs direkt aus der Browser-Symbolleiste gespeichert, angezeigt und verwaltet werden können — kein App-Wechsel nötig.

Funktionsweise

Aktuellen Tab speichern

Klick auf das Erweiterungs-Symbol speichert die URL des aktiven Tabs mit einem Klick. URLs werden in Chromes lokalem Speicher abgelegt, sodass sie sitzungsübergreifend erhalten bleiben.

Manuelle Eingabe

Beliebige URLs in das Eingabefeld eingeben oder einfügen — zum Speichern von Links aus Dokumenten, E-Mails oder Notizen außerhalb des Browsers.

Anzeigen & Löschen

Alle gespeicherten URLs werden als klickbare Links aufgelistet. Doppelklick auf den Löschen-Button leert die gesamte Liste.

Technologien

  • JavaScript (ES6) — Kernlogik und DOM-Manipulation
  • Chrome Extensions API — chrome.tabs und chrome.storage.local
  • HTML & CSS — Popup-UI der Erweiterung
  • Scrimba-JavaScript-Kurs — Lernrahmen
