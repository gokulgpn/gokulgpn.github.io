Digitale Visitenkarte

HTMLCSSResponsives DesignNetlify

Überblick

Eine kompakte persönliche Visitenkarten-Website, entwickelt zum Üben von HTML/CSS-Layout, visueller Hierarchie und einfacher responsiver Darstellung.

Was ich gebaut habe

Karten-Layout

Gestaltung einer fokussierten Ein-Karten-Oberfläche, die ein Profilbild, Berufsbezeichnung, Standort und wichtige Kontaktaktionen präsentiert.

Responsives Styling

Übung in Abstands-, Typografie-, Farb- und Größengestaltung, damit die Karte über alle Browser-Breiten hinweg sauber funktioniert.

Öffentliches Deployment

Veröffentlichung der statischen Website über Netlify aus einem GitHub-Repository, um das Projekt leicht teilbar zu machen.

Tools & Technologien

  • HTML für die Struktur
  • CSS für Layout und visuelle Gestaltung
  • GitHub für das Repository-Hosting
  • Netlify für das Live-Deployment
